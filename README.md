Support code and images for [Batch convert images to PDF with Python by using Pillow or img2pdf](https://solarianprogrammer.com/2019/06/12/batch-convert-images-to-pdf-with-python-using-pillow-or-img2pdf/)

Usage examples

```shell
python convertor_0.py

python convertor_1.py -prefix="imgs_png/" -max_range=3 -output=tst_from_png.pdf
python convertor_1.py -prefix="imgs_jpg/" -suffix=.jpg -max_range=3 -output=tst_from_jpg.pdf

python convertor_2.py -prefix="imgs_jpg/" -max_range=3 -output=tst_from_jpg_2.pdf
```
